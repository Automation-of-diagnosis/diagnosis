Архитектура приложения:

my_app.py - модуль, в котором осуществляется создание FLASK приложения, 
создание экземпляра класса FLASK

config.py - модуль с настройками приложения

funcs.py - модуль или пакет с функциями или классами, 
в которых описана основная логика работы, бизнес-логика

models.py - модуль или пакет для доступа к базе данных

requriments.txt - файл с командой FREZEE, какие пакеты установлены

runserver.py - модуль, в котором происходит запуск 
приложения на тестовом локальном сервере

template - папка содержащая html страницы

tests - папка с тестами модуля

utils.py - модуль с утилитными функциями, которые используются 
из различных модулей

views.py - модуль в котором описаны функции и соответствующие им url'ы приложения

venv_app - папка для виртуального окружения приложения

.gitignore - файл с указанием файлов, которые не комитятся
